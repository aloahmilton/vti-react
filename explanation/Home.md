# Home.tsx Component

The `Home` component serves as the landing page for the store, providing a professional and welcoming first impression.

## Responsibility
- **Marketing Display**: Showcases the brand and provides a high-level overview of the store's value proposition.
- **Featured Products**: Displays a subset of products (the first 3) to entice users immediately.
- **Informational Content**: Uses cards to explain delivery, security, and support policies.

## Props
- `featuredProducts`: An array of 3 products to display.
- `onAddToCart`: A function passed from `App.tsx` to handle "Add to Cart" actions directly from the home page.
